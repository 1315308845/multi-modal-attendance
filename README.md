项目简介：开发了一款多模态考勤系统，支持 IC 卡、指纹与人脸识别，考勤信息通过微信小程序展示。
所用技术栈：STM32F407、FreeRTOS、ESP32、Raspberry Pi、SPI、UART、HTTP、OpenCV
项目工作：
1. STM32F407VET6 作为主控，通过 SPI 读写 IC 卡扇区，通过 UART 与指纹模块通信；树莓派 Linux 负责图像采集、训练与
识别，结果通过串口返回主控。
2. 构建基于结构体的轻量级本地特征数据库，实现快速身份检索。
3. 考勤数据通过 UART 发送至 ESP32，ESP32 同步 NTP 时间后通过 HTTP 上报云端；小程序调用云函数展示考勤信息。
4. 将 STM32 的识别、检索、通信等模块划分为独立 FreeRTOS 任务，实现多线程调度。
